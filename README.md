# Salafah Menu Starter (صور + أسعار)

## كيف تركّبه داخل مشروع Next.js (App Router)
1) انسخ المجلدات:
- `app/page.js`
- `components/MenuSection.jsx`
- `data/menuData.js`
- مجلد الصور `public/menu/*`

2) شغّل:
```bash
npm install
npm run dev
```

## تعديل الأسعار / الأسماء
- من الملف: `data/menuData.js`
